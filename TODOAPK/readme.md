# ✅ Todo List App (Node.js + Express + MongoDB + EJS)

A full-stack **Todo List Application** built with **Node.js, Express, MongoDB (Mongoose), and EJS**.  
Users can create tasks, mark them as done, undo them, and delete tasks.

---
![](./public/image.png)

## 🚀 Features

- ➕ Add new tasks  
- ✅ Mark task as done  
- 🔄 Undo completed task  
- ❌ Delete task  
- 📊 Task status (Pending / Done)  
- 🎨 Styled with Tailwind CSS  

---

## 🛠️ Tech Stack

- **Backend:** Node.js, Express  
- **Database:** MongoDB (Mongoose)  
- **Frontend:** EJS + Tailwind CSS  
- **Architecture:** MVC Pattern
